# Task Manager (Java)

A console-based Task Manager application built using Java.  
This project allows users to manage their daily tasks efficiently using a simple menu-driven interface.

## Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Console-based user interface

## Technologies Used
- Java
- Object Oriented Programming (OOP)
- Java Collections
- IntelliJ IDEA

## How to Run
1. Clone the repository
2. Open the project in IntelliJ IDEA
3. Run the Main.java file
